Welcome to my game! This game is called a ball game also known as ping-pong game in which two players hit a lightweight ball back and forth across a black screen using small paddles. A point is scored when a player fails to return the ball to the opponent side. In this game, a player will always face a computer(AI) and there will be a chance to win the AI but the player should coup up with the ball speed since the ball's speed increases at each bounce. So, if the player don't lose at high speed,there will be a better chance to win the AI. So, as you can see,it is much similar to table tennis game except it is not on the feild😜. Thanks a lot for being with us!

To pause the game : press space
To start the game : press space
To move the paddles : use the mouse

This game's front end is javascript which is based on ECMAScript 6 and I have used local storage to keep track of the highest score so far.
